# Manual Tecnico - Conecta Portal

## Objetivo
Unificar integraciones tributarias y previsionales.

## Rutas clave
- /administracion/sii
- /administracion/previred
- /conexiones
- /administracion/dashboard-tributario

## Permisos
- admin_sii
- admin_previred
- admin_conexiones
- admin_dashboard_tributario

## Operaciones
- Configurar credenciales SII y certificados.
- Configurar enlace Previred.
- Revisar estado de conectores y errores de sincronizacion.

## Soporte rapido
1. Verificar token y estado de API externa.
2. Revisar si existe contexto de auditoria activo.
3. Confirmar permiso ver del modulo correspondiente.
